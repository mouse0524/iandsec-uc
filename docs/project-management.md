# 项目管理

项目管理用于登记已签单项目，并把项目与现有工单按客户公司联动。

## 权限

- 管理员、客服、技术可进入项目管理。
- 客服、技术可新增项目、修改项目、流转状态、指派负责人和维护运维记录。

## 配置

系统设置的“工单配置”中新增三组动态标签：

- 项目产品：项目“使用产品”的可选值，默认 `安得卫士`。
- 项目状态：项目状态流转的可选值，默认 `待实施`、`已完工`。
- 运维类型：运维记录类型，默认 `迁移库`、`重做系统`、`运维`、`其他`。

## 联动规则

- 项目列表的问题单数量来自现有工单：工单 `company_name` 等于项目名称，且跟踪为 `现网问题`。
- 项目列表的需求单数量来自现有工单：工单 `company_name` 等于项目名称，且跟踪为 `现网需求`。
- 运维单数量来自项目运维记录。
- 点击问题单/需求单数量会跳转到“我的工单”并带上客户公司与跟踪筛选；点击运维单数量会跳转到运维记录并带上项目筛选。

## 数据表

- `project`：项目主表。
- `project_activity`：项目运维记录，覆盖迁移库、重做系统、运维等必要信息。
